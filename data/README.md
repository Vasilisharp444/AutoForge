# Data Directory

## Sample Data

`sample_NQ.csv` contains 5,000 synthetic NQ 21-range bars for demo and testing purposes. It was generated by `generate_sample.py` using a mean-reverting random walk — no real market data is included.

Characteristics:
- Price range: ~18000-19000 (realistic NQ levels)
- High-Low spread: ~21 points per bar (range bars)
- Volume: 50-500 per bar
- Timestamps: Pacific Time, spanning ~10 trading days

To regenerate: `python data/generate_sample.py`

## Bring Your Own Data

For real analysis, export your own OHLCV data from NinjaTrader, Sierra Chart, or any platform. Place CSV files here with columns:

```
DateTime, Open, High, Low, Close, Volume
```

DateTime format: `YYYY-MM-DD HH:MM:SS`
